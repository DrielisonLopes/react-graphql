# Full-stack (GraphQL + Apollo)

A simple React application using GraphQL and Apollo Client to manage users.

## 📌 Features

- List users
- Get user by ID
- Create new user
- Apollo Client integration
- Custom hooks (useUsers, useUserById, useCreateUser)
- Apollo cache update (no refetch)
- Component-based architecture
- TypeScript support

## 🧠 Concepts applied

- GraphQL queries and mutations
- Apollo cache management
- Custom React hooks
- Separation of concerns
- React.memo optimization

## 🛠 Tech Stack

- React
- TypeScript
- Apollo Client
- GraphQL

## ▶️ Running the project

```bash
npm install
npm run dev
```
